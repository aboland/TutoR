Objects

Single Values

You can assign values to a name, this is known as creating an object. The <- operator assigns the value to the object name.

Calling the object name will return the value.

my_object <- 4

my_object

Vector of values

You can also assign a set (or vector) of values to a name.

A vector of values is written using c(var1, var2, var3).

my_vector <- c(1, 4, 6)

my_vector

Helpfiles

There are help files for all functions. The help files will explain what the function should do and what arguments should/can be given. Help files can be accesed using ?

?mean

The helpfile explains the mean function and the available arguments. There is an optional argument na.rm, this removes NA values. A NA value is a blank entry

What happens when we try to calculate the mean of a set of variables which contain an NA entry?

An additional arguments can fix this, use the helpfile to fix the code below?

my_vector2 <- c(3, 4, 5, NA)
my_vector2

mean(my_vector2)
mean(my_vector2, na.rm = TRUE)

Tidyverse

We will cover the tidyverse in more detail in the coming weeks.
The pipe concept %>%, which passes the returned object on the left into the function on the right. The object is always passed as the first argument of the function.

my_vector2 <- c(3, 4, 5, NA)

my_vector2 %>% mean
my_vector2 %>% mean(na.rm = TRUE)
